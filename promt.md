# AI Usage Documentation

This project was primarily developed by me. AI was used only to understand specific concepts, clarify implementation approaches, and resolve a few development issues. The overall project structure, UI, component integration, and final implementation were completed by me.

## AI Assistance

### 1. React Hook Form (Learning)

Used AI to understand the basics of React Hook Form, including:

- register()
- handleSubmit()
- formState.errors
- trigger()
- getValues()

---

### 2. Zod Validation

Asked AI to understand how to create a validation schema using Zod and how to integrate it with React Hook Form.

---

### 3. Multi-Step Form Flow

Used AI to understand how data can persist while navigating between multiple form steps.

---

### 4. Progress Bar

Used AI to understand the logic for creating a step progress indicator in a multi-step form.

---

### 5. Debugging

Used AI to debug a few issues related to:

- Form validation
- Step navigation
- Back button state retention

---

## AI Usage Statement

AI was used only as a learning and debugging assistant to understand React Hook Form, Zod, multi-step form architecture, and to resolve specific implementation issues. The project design, UI development, component creation, styling, integration, testing, and overall implementation were completed independently by me.
